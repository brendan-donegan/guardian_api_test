Guardian API tests
==================

A collection of tests for the RESTful API provided by http://content.guardianapis.com

Scope of testing
================

- Basic Queries
- From date
- To date
- Page size
- Current page

Running tests
=============

To run the tests:

1. Create a virtual environment
1. ```pip install -r requirements.txt```
1. set the ```API_KEY``` environment variable to your API key
1. ```pytest```
